🖨️ How to Map a Printer (Windows)

        
    
          
  1. Open the Start Menu → type Run and hit Enter.
    2. 
          
  2. Type \\[PrinterServerName] (e.g., \\printer01).
    3. 
          
  3. Press Enter to view shared printers.
    4. 
          
  4. Double-click the desired printer to install it.
    5. 
          
  5. Alternatively, go to Devices and Printers → Add a printer.
    6.

📡 Antenna Placement Tips

        
    
          
  • Keep vertical and away from metal structures.
    • 
          
  • Elevate for better Line of Sight (LOS).
    • 
          
  • Use poles or trees to extend height when needed.
